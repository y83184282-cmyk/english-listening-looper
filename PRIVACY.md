# Privacy Policy

English Listening Looper does not collect, sell, share, or transmit user data to any remote server.

The extension stores the following data locally in the browser with `chrome.storage.local`:

- Sentence node timestamps for the current page.
- A hashed page identifier used to associate saved sentence nodes with the current page.
- Shortcut preferences.
- Overlay UI preferences, including size and shortcut panel collapsed state.

The extension does not store the raw page URL or page title. It uses the current page address only inside the browser to compute the page-specific hash.

This data stays on the user's device unless the user or browser sync settings separately copy extension storage. The extension does not use analytics, advertising, tracking pixels, remote code, or external network requests.

Users can remove the stored data by removing the extension or clearing extension site data from the browser.
